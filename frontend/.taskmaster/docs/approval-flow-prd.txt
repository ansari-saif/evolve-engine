# Overview  
Add approval flow to GenerateDailyTasksDialog allowing users to review, edit, and approve AI-generated tasks before creation.

# Core Features  
- **Task Preview**: Display generated tasks in editable list after API call
- **Task Editing**: Allow modification of task details before creation  
- **Approval Workflow**: Two-step process: generate → review/edit → create
- **Bulk Creation**: Create all approved tasks at once

# User Experience  
- User opens dialog, sets energy/phase, clicks "Generate Tasks"
- Generated tasks appear in editable list view
- User edits task titles, descriptions, priorities
- User clicks "Create Tasks" to bulk create all tasks
- Dialog closes, tasks appear in main list
